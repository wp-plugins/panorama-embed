== Description ==

Panorama Embed allows you to show panoramas or virtual tours in you blog, either inline, or linked by a picture.
See demo and description at <a href="http://gpwrite.com/panorama-embed-for-wordpress/">Wordpress Panorama Plugin</a>


== Installation ==

Upload the Panorama Embed plugin to your blog, and activate it. Set the size of your embedded panoramas on the Admin Page.
Upload your panoramas or virtual tours, each in their own directory, or use existing panoramas (if they have their own directory)
Add shortcode panoembed in your posts.

1, 2, 3: You're done!

== Frequently Asked Questions ==

= Can I use any type panorama or virtual tour? =

Probably yes, because the panorama is either in an iframe, or new window.

= Can I use the plugin with the panoramas I already uploaded? =

If the panoramas or virtual tours are each in their own directory, yes.
